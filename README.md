# RealTime-Holistic-Landmark-Detection-Using-MediaPipe

This project demonstrates real-time facial and hand landmark detection using MediaPipe Holistic and OpenCV.
It captures live webcam video, detects face mesh contours and both hand landmarks, and displays them along with real-time FPS.

## Built with:

🐍 Python

🎥 OpenCV

🤖 MediaPipe

## 🚀 Features

✅ Real-time webcam capture

✅ Face Mesh contour detection

✅ Left & Right hand landmark detection

✅ FPS calculation and display

✅ Clean and customizable drawing styles

## 📌 How It Works

Captures video from the default camera.

Converts BGR → RGB (required by MediaPipe).

Processes the frame using MediaPipe Holistic Model.

## Draws:

Face mesh contours

Right hand landmarks

Left hand landmarks

Calculates and displays FPS.

Press ESC to exit.
